# /sindri

Available commands:

- `/sindri:init` — Set up an autonomous improvement loop (explore project, design evaluate function, scaffold .sindri/)
- `/sindri:loop` — Start or resume the continuous experiment loop (agent iterates forever until stopped)
- `/sindri:cycle` — Run exactly one experiment cycle and stop (for delayed feedback domains)

Communicate with the user in their language. Detect from their messages or system locale.

Tell the user about these commands and help them pick the right one.
